# Etiquetas Principales

Las etiquetas principales de los archivos RSS:

* Etiqueta 'rss'
* Etiqueta 'title'
* Etiqueta 'channel'
* Etiqueta 'description'
* Etiqueta 'link'
* Etiqueta 'category'

Las etiquetas principales de los archivos ATOM:

* Etiqueta 'feed'
* Etiqueta 'title'
* Etiqueta 'entry'
* Etiqueta 'link'
* Etiqueta 'id'
* Etiqueta 'update'
* Etiqueta 'author'
* Etiqueta 'name'
